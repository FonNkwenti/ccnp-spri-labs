# Lab 05: BGP Communities and FlowSpec — Topology

## Summary

This topology spans **four autonomous systems** connected in a hub-and-spoke arrangement
centered on AS 65100 (the SP core). It contains **7 routers** and **8 physical links**.

```
AS 65001 (R1 CE)
    |
    |-- L1 eBGP primary   --> R2 PE East-1 --|
    |-- L2 eBGP backup    --> R3 PE East-2 --|
                                             |-- L3/L4 OSPF/iBGP --> R4 P/RR
                                             L6 IGP R2<->R3        |
                                                                    |-- L5 OSPF/iBGP --> R5 PE West
                                                                                         |-- L7 eBGP --> R6 (AS 65002)
                                                                                         |-- L8 eBGP+FlowSpec --> R7 (AS 65003)
```

### Layout Description

| Zone | Devices | Position |
|------|---------|----------|
| AS 65001 | R1 (CE) | Top-left |
| AS 65100 | R2 (PE East-1), R3 (PE East-2), R4 (P/RR), R5 (PE West) | Center |
| AS 65002 | R6 (External SP Peer) | Top-right |
| AS 65003 | R7 (FlowSpec Originator) | Bottom-right |

R4 is the iBGP Route Reflector for all AS 65100 PEs (R2, R3, R5).
OSPF area 0 runs across the AS 65100 core on all internal links and loopbacks.
R5 and R7 are CSR1000v (IOS-XE) to support the BGP FlowSpec SAFI.

## EVE-NG Import Instructions

1. Open the EVE-NG web UI and log in.

## Starting the Lab

1. Open the lab in EVE-NG.
2. Select all nodes (Ctrl+A) and click **Start** (or right-click > Start).
3. Wait approximately 2-3 minutes for all IOSv nodes to boot and 4-5 minutes for CSR1000v
   nodes (R5, R7) to reach a usable prompt.
4. Run the setup script to push the pre-configured base configuration:

   ```bash
   python labs/bgp/lab-05-communities-flowspec/setup_lab.py --host <eve-ng-ip>
   ```

5. Console into any device via:

   ```bash
   telnet <eve-ng-ip> <console-port>
   ```

   Console port numbers are shown in the EVE-NG UI node properties panel.

## Exporting the Lab

To export the lab as a `.unl` archive for sharing or backup:

1. In the EVE-NG web UI, right-click the lab in the lab list.
2. Select **Export** and save the `.unl` file.
3. Alternatively, from the EVE-NG shell, the lab files are located at:

   ```
   /opt/unetlab/labs/<your-folder>/<lab-name>.unl
   ```

To export just the node startup configs:

```bash
# From EVE-NG shell
python /opt/unetlab/scripts/export-configs.py --lab <lab-name>.unl
```
